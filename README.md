# Export to mp4

https://github.com/mapbox/mapbox-gl-js/blob/main/debug/video-export.html
https://ffmpeg.org/
`ffmpeg -i my_video.mp4 my_video_compressed.mp4`

# Export to gif
```bash
ffmpeg -i input.mp4 \
    -vf "fps=10,scale=320:-1:flags=lanczos,split[s0][s1];[s0]palettegen[p];[s1][p]paletteuse" \
    -loop 0 output.gif
```
